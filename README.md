# Smart Task Management System

A web-based application designed to help users efficiently manage their daily tasks, track progress, and improve productivity through an intuitive and user-friendly interface.

## Features

* Create, update, and delete tasks
* Set task priorities
* Track task status (Pending, In Progress, Completed)
* User-friendly dashboard
* Task organization and management
* Responsive design

## Technologies Used

* Python
* Flask
* HTML
* CSS
* JavaScript
* SQLite/MySQL (if applicable)

## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/Smart_Task_Management_System.git
```

2. Navigate to the project directory:

```bash
cd Smart_Task_Management_System
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python app.py
```

5. Open your browser and visit:

```
http://127.0.0.1:5000
```

## Usage

* Add new tasks with details and priority levels.
* Update task information as needed.
* Mark tasks as completed.
* Monitor task progress through the dashboard.
